# Pulse.

**Projeto de monitoramento de bem-estar e consciência corporal**  

O Pulse. é uma aplicação web que permite acompanhar check-ins diários, métricas físicas e registros de dor, ajudando o usuário a desenvolver consciência corporal e acompanhar sua evolução ao longo do tempo.

## Funcionalidades

- Cadastro e login de usuários com autenticação segura
- Registro de check-ins diários
- Acompanhamento de métricas físicas (energia, sono, humor, fadiga)
- Timeline com histórico de check-ins e registros de dor
- Interface responsiva e intuitiva
- Proteção de dados via RLS no banco (cada usuário só acessa seus próprios dados)

## Tecnologias Utilizadas

- Frontend: React 18 + TailwindCSS + Vite
- Backend: Supabase (PostgreSQL + Auth)
- Gerenciamento de estado e dados: React Query
- Autenticação: Context API + Supabase Auth
- Deploy: GitHub Pages
